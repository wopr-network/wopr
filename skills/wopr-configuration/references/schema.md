# Config Schema (Top-Level Keys)

The central config includes these top-level sections:

- `daemon`: `port`, `host`, `autoStart`
- `anthropic`: `apiKey`
- `oauth`: `clientId`, `clientSecret`, `redirectUri`
- `discord`: `token`, `guildId`
- `discovery`: `topics`, `autoJoin`
- `plugins`: `autoLoad`, `directories`, `data` (plugin-specific config)

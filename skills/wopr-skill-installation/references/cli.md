# Skill CLI Commands

## List skills
`wopr skill list`

## Install skills
`wopr skill install <url|slug> [name]`

## Create/remove skills
- `wopr skill create <name> [desc]`
- `wopr skill remove <name>`

## Search registries
`wopr skill search <query>`

## Cache
`wopr skill cache clear`

## Skill registries
- `wopr skill registry list`
- `wopr skill registry add <name> <url>`
- `wopr skill registry remove <name>`

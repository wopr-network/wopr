# WOPR Plugin Management

Use this skill to manage plugin lifecycle (install/enable/disable/remove) and plugin registries.

## Quick workflow

1. Install a plugin from npm, GitHub, or a local path.
2. Enable or disable the plugin as needed.
3. Search for plugins and manage plugin registries.

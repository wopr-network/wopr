# Session CLI Commands

Use these commands to manage sessions:

- `wopr session create <name> [context]`
- `wopr session inject <name> <message>`
- `wopr session list`
- `wopr session show <name> [--limit N]`
- `wopr session delete <name>`

The CLI also supports direct peer injection (P2P) via:

- `wopr inject <peer>:<session> <message>`
